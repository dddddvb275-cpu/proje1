# Province Search Project

This project is a web application developed to practice modern JavaScript, asynchronous operations, and dynamic DOM manipulation. It serves as a milestone in my software development journey.

## 🚀 About the Journey
Just a month ago, I was building simple HTML forms. Today, I am architecting modular structures, fetching data from APIs, and implementing real-time search functionality. This project is a documentation of my progress and my dedication to learning software development.

## 🛠 Features
- **API Integration**: Fetches province data dynamically.
- **Modular Structure**: Developed with a clean, modular code approach.
- **Responsive Layout**: Uses CSS Flexbox for organized and mobile-friendly UI.
- **Real-time Search**: Allows users to filter provinces instantly via input.
- **Dynamic Rendering**: Displays data in a card-based layout.

## 💻 Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Flexbox layouts for design.
- **JavaScript (ES6+)**: Asynchronous programming (`fetch`, `async/await`) and array methods (`filter`).

## 🎯 Main Goal
My primary goal with this project is to track my own growth. Every line of code here represents a step forward in understanding how modern web applications work. I am learning, building, and moving forward!
